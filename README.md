## 기술 스택

- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Google Fonts (Oswald, Noto Sans KR)
- GitHub Pages (배포)

---

## 프로젝트 구조

```
Web-portfolio/
├── index.html                  # 메인 페이지
├── css/
│   ├── variables.css           # CSS 변수, 리셋, 공통 스타일
│   ├── animations.css          # 애니메이션 (fadeUp, reveal, 커서)
│   ├── nav.css                 # 상단 네비게이션
│   ├── hero.css                # 히어로 섹션
│   ├── profile.css             # 프로필 + 소개글
│   ├── skills.css              # 스킬 카드
│   ├── experience.css          # 경험 + 교육 타임라인
│   ├── projects.css            # 프로젝트 카드 그리드
│   └── contact.css             # 연락처 + 푸터
├── js/
│   ├── typing.js               # 히어로 타이핑 애니메이션
│   └── scroll-reveal.js        # 스크롤 페이드인 애니메이션
├── images/
│   ├── myphoto.jpg             # 프로필 사진
│   └── projects/               # 프로젝트 이미지
│       ├── hbot.png
│       ├── elephant-bike.png
│       ├── atmega4809.jpg
│       └── ecg.jpg
└── .github/
    └── workflows/
        └── deploy.yml          # GitHub Pages 자동 배포
```

## 섹션 구성

| 섹션 | 설명 |
|------|------|
| **Profile** | 기본 정보, 연락처 |
| **Skills** | 언어, MCU/하드웨어, 개발 도구 |
| **Experience** | 학부 연구생, 석사 펌웨어 개발 연구원 |
| **Education** | 의공학 석사 / 의료시스템공학 학사 |
| **Projects** | 연구 프로젝트 2개, 사이드 프로젝트 2개 |
| **Contact** | 이메일, 전화, GitHub, 블로그 |
